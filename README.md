# Plantobot
a 3d print bot for home plant, arduino decorative

#################################################

this bot is a simple arduino weather station 3d printable.
Electronics are composed by :
- 1 arduino micro
- 1 moisture sensor
- 1 DTH11 sensor
- 1 LCD basic screen
- 1 10k plot
- 1 ligth sensor
